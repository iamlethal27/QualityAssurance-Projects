# SauceDemo QA Project

## 📌 Project Overview

This project demonstrates manual testing of the SauceDemo website including test case design, execution, and bug reporting.

## 🔧 Tools Used

* Manual Testing
* Postman (API Testing - upcoming)
* Jira (Bug Tracking)

## 📂 Modules Covered

* Login
* Product
* Cart
* Checkout
* Logout

## ✅ Testing Types

* Functional Testing
* UI Testing
* Edge/Negative Testing
* Security Testing

## 🧪 Test Artifacts

* Test Scenarios
* Test Cases
* Bug Reports (Excel + Jira)

## 🐞 Bug Reporting

Bugs are reported with:

* Severity & Priority
* Steps to Reproduce
* Expected vs Actual Result
* Screenshots

## 📸 Jira Evidence

Jira bug screenshots are attached in Bug-Reports folder.
